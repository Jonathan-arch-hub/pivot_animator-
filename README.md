# pivot_animator 🚀

محرك رسوم متحركة ثنائي الأبعاد (2D) ومفتوح المصدر مخصص لإنشاء حركات الأشكال الخيطية (Stick-figures) والـ Pivot. المشروع مكتوب بالكامل بلغة **C++** ويعتمد على **CMake** كنظام بناء أساسي لتسهيل التطوير والتثبيت عبر المنصات.

### ✨ الميزات الأساسية والهيكلية:
* **Canvas & Timeline:** إدارة النوافذ، لوحة الرسم (`canvas.h`)، والشريط الزمني للـ keyframes عبر (`timeline.h`).
* **Interpreter & Exporter:** مفسر داخلي للبيانات (`interpreter.h`) مع دعم تصدير الحركات وصناعة المخرجات النهائية (`exporter.h`).
* **Code Panel & Core:** واجهة تحكم متكاملة للأدوات (`codepanel.h`) مع هيكلة بيانات مروّسة وعالية الكفاءة (`types.h`).
* **سهولة البناء:** سكربت أتمتة جاهز للبناء والتثبيت السريع (`build_and_install.sh`).



=========================English==============================
# pivot_animator 🚀

An open-source, cross-platform 2D stick-figure animation engine inspired by Pivot Animator. Built entirely in modern **C++** and managed with **CMake** for clean, scalable, and modular development.

### ✨ Key Architecture & Features:
* **Canvas & Timeline:** Interactive rendering workspace (`canvas.h`) coupled with a keyframe-based timeline system (`timeline.h`).
* **Interpreter & Exporter:** Robust internal backend for interpreting animation data (`interpreter.h`) and exporting to production-ready formats (`exporter.h`).
* **UI & Core Control:** Dedicated control dashboard (`codepanel.h`) and structured, highly optimized custom data types (`types.h`).
* **Hassle-Free Setup:** Includes an automated build and installation script (`build_and_install.sh`) for quick deployment.
